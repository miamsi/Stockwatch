# 📈 Institutional Equity Intelligence Dashboard

A multi-factor decision support system for equity analysis, integrating:

- 📊 Fundamental Analysis (Health Score Model)
- 🌍 Macro Correlation (IHSG & USD/IDR)
- 📰 NLP Sentiment Analysis (Real-time News)
- 🔮 Scenario-Based Valuation Modeling

## 🚀 Live App
[Click here to open the dashboard](YOUR_STREAMLIT_LINK)

## 🧠 Key Features

### 1. Fundamental Engine
- P/E Ratio, Profit Margin, Debt-to-Equity
- Proprietary Health Score (0–100)

### 2. Macro Analysis
- Stock vs IHSG vs USD/IDR performance
- Correlation matrix

### 3. Sentiment Analysis
- NLP-based scoring using financial headlines
- Weighted sentiment model

### 4. Scenario Simulation
- Growth vs Risk stress testing
- Beta-adjusted valuation

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- yFinance
- TextBlob

## 🎯 Purpose
This project simulates how institutional investors and consultants analyze equities using multi-factor models.
